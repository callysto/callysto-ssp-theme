# Callysto SimpleSAMLphp Theme

This repo contains the theme files for the SimpleSAMLphp portion of Callysto.

## Installation

1. Clone this directory to `/var/simplesamlphp-x.xx/modules/callysto`.
2. Edit `/var/simplesamlphp-x.xx/config/config.php` and set `theme.use` to `callysto:callysto`:

```
'theme.use' => 'callysto:callysto',
```
